## 향후 구현 항목

### 1. 평가 모듈 (우선순위: 높음)

#### 1.1 자동 점수 계산
- [ ] ScoringCriteria 기반 자동 점수 계산 로직
- [ ] 평가 기준 관리 UI (운영자)
- [ ] 지원서별 점수 계산 및 저장

#### 1.2 증빙 첨부 가감점 자동화
- [ ] `proof_required_level == OPTIONAL`인 항목에서:
  - 증빙 첨부 시 → 가점 부여
  - 미첨부 시 → 감점 처리
- [ ] ProjectItem에 `proof_bonus_score` 필드 추가 검토
- [ ] 평가 시 자동 반영 로직

#### 1.3 평가 결과 관리
- [ ] 지원자별 총점 조회 UI
- [ ] 선발 기준(컷오프) 설정
- [ ] 일괄 선발/탈락 처리

## 구현 우선순위

1. **즉시**: 평가 모듈 (점수 계산, 선발 처리)
2. **단기**: Excel 다운로드, 이메일 안정화
3. **중기**: SMS, 6개월 리마인드, Excel 업로드
4. **장기**: 본인인증, 고급 통계
