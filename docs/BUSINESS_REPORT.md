## 1. 시스템 개요

### 1.1 배경 및 필요성

#### 기존 문제점

**과제별로 다른 평가 기준 관리의 어려움**
- 각 과제마다 요구하는 역량이 다름 (비즈니스 코칭, 청소년 코칭, 라이프 코칭 등)
- Excel 또는 수기로 설문지 작성 → 시간 소요 및 오류 발생
- 과제마다 새로운 설문지 제작 필요 (평균 2시간 소요)

**수작업 평가의 한계**
- 제출된 서류를 일일이 확인하며 점수 산정
- 평가자마다 기준이 다를 수 있음 (주관적 평가)
- 증빙 서류 누락 확인 어려움

**코치 입장의 불편함**
- 동일한 정보를 과제마다 반복 입력
- 증빙 서류를 어디에 첨부해야 할지 불명확
- 제출 전 검증 기능 없어 오류 발생 가능

#### 해결 방안

**"레고 블록처럼 조립하는 설문 시스템"**

마치 레고 블록을 조립하듯, **미리 만들어진 평가 항목들을 선택**하여 과제별 맞춤 설문을 빠르게 구성할 수 있는 시스템입니다.

- **13개의 표준 역량 항목** (학력, 자격증, 경력, 코칭 분야별 이력 등)
- **5가지 입력 방식** (텍스트, 숫자, 선택형, 파일 첨부 등)
- **클릭 몇 번으로 설문 완성** (드래그 앤 드롭 방식)
- **자동 점수 검증** (총 100점 맞춤)

### 1.2 시스템 목표

1. **업무 효율성 극대화**: 설문 구성 및 평가 시간 70% 이상 단축
2. **평가 공정성 확보**: 표준화된 기준으로 객관적 평가
3. **사용자 편의성 향상**: 직관적인 UI로 누구나 쉽게 사용
4. **확장 가능성**: 향후 새로운 역량 항목 추가 용이
