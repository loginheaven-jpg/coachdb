## 2. 접속 방법

### 2.1 접속 정보

**웹사이트 주소**:
```
https://coachdb-pilot.yourdomain.com
```

**테스트 계정**:

| 역할 | 이메일 | 비밀번호 | 용도 |
|------|--------|----------|------|
| **관리자 1** | admin1@test.com | Pilot2025! | 설문 구성, 평가 |
| **관리자 2** | admin2@test.com | Pilot2025! | 설문 구성, 평가 |
| **코치 1** | coach1@test.com | Pilot2025! | 과제 응모 |
| **코치 2** | coach2@test.com | Pilot2025! | 과제 응모 |
| ... | ... | ... | (총 10명) |

⚠️ **주의사항**:
- 테스트용 계정이므로 실제 개인정보를 입력하지 마세요
- 비밀번호는 첫 로그인 후 변경해주세요

### 2.2 권장 환경

**PC (권장)**:
- 브라우저: Chrome, Edge, Safari 최신 버전
- 화면: 1920x1080 이상
- 인터넷: 10Mbps 이상

**모바일 (제한적 지원)**:
- iOS 14 이상, Android 10 이상
- 일부 기능은 PC에서만 사용 가능

### 2.3 첫 로그인

1. 웹사이트 접속
2. 로그인 버튼 클릭
3. 제공받은 이메일/비밀번호 입력
4. "비밀번호 변경" 안내 따라 변경
5. 메인 화면 확인

## 3. 테스트 시나리오

### 3.1 관리자 시나리오

#### 시나리오 A: 설문 구성하기 (소요 시간: 15분)

**목표**: 새로운 과제를 만들고 설문을 구성합니다.

**단계**:

```
1️⃣ 과제 생성
   - 좌측 메뉴 "과제 관리" 클릭
   - "새 과제 만들기" 버튼 클릭
   - 과제 정보 입력:
     • 과제명: [파일럿] 비즈니스 코칭 모집
     • 모집 기간: 2025-11-10 ~ 2025-11-30
     • 모집 인원: 5명
   - 저장

2️⃣ 설문 구성
   - 생성한 과제 클릭
   - "설문 구성" 버튼 클릭
   - 항목 선택:
     ☑ 코칭 관련 최종학력 (10점)
     ☑ KCA 자격증 (15점)
     ☑ 총 코칭 경력 (10점)
     ☑ 비즈니스코칭 이력 (20점)
     ☑ 커리어코칭 이력 (15점)
     ☑ 전문 분야 (5점)
     • 커스텀 질문 추가: "귀하의 코칭 철학은?" (25점)
   - 총점 확인: 100/100 ✓

3️⃣ 증빙 설정
   - 각 항목의 증빙 레벨 설정:
     • 학력: 선택
     • 자격증: 필수
     • 이력: 선택

4️⃣ 미리보기
   - "미리보기" 버튼 클릭
   - 코치가 보게 될 화면 확인
   - 문제 없으면 "저장" 클릭

5️⃣ 과제 발행
   - 상태를 "모집중"으로 변경
```

**체크포인트**:
- [ ] 설문 구성 화면이 직관적인가?
- [ ] 100점 검증이 제대로 작동하는가?
- [ ] 미리보기가 실제와 동일한가?
- [ ] 저장이 정상적으로 되는가?

**예상 소요 시간**: 약 15분

---

#### 시나리오 B: 응모자 평가하기 (소요 시간: 10분)

**목표**: 코치가 제출한 응모서를 평가합니다.

**단계**:

```
1️⃣ 응모 목록 확인
   - 과제 상세 페이지
   - "응모자 관리" 탭 클릭
   - 제출된 응모서 목록 확인

2️⃣ 응모서 상세 보기
   - 응모자 클릭
   - 입력된 내용 확인:
     • 학력 정보
     • 자격증 (복수 입력 확인)
     • 경력 및 이력
     • 첨부 파일 다운로드

3️⃣ 자동 점수 확인
   - 시스템이 자동으로 산정한 점수 확인
   - 채점 기준 충족 여부 확인

4️⃣ 수동 점수 입력
   - 자유 응답 항목 평가
   - 증빙 서류 확인 후 검증 상태 변경
   - 최종 점수 입력

5️⃣ 평가 완료
   - "평가 완료" 버튼 클릭
   - 코치에게 알림 발송
```

**체크포인트**:
- [ ] 응모 내용이 깔끔하게 보이는가?
- [ ] 첨부 파일을 쉽게 열람할 수 있는가?
- [ ] 자동 점수가 정확한가?
- [ ] 평가 작업이 빠르고 편리한가?

**예상 소요 시간**: 건당 약 10분

### 3.2 코치 시나리오

#### 시나리오 C: 과제 응모하기 (소요 시간: 30분)

**목표**: 관심 있는 과제에 응모합니다.

**단계**:

```
1️⃣ 과제 찾기

   - 관심 과제 클릭

2️⃣ 과제 정보 확인
   - 과제 상세 내용 읽기
   - 모집 기간, 인원 확인
   - 평가 항목 확인
   - "응모하기" 버튼 클릭

3️⃣ 설문 작성
   📚 학력 항목
   - 학위 선택: [석사 ▼]
   - 전공명 입력: [상담심리학]
   - 증빙 파일: [파일 선택] (선택사항)

   🏆 자격증 항목
   - 자격증 1:
     • 명칭: [KAC 전문코치]
     • 증빙: [파일 선택] ⚠️ 필수
   - "자격증 추가" 버튼으로 여러 개 등록 가능

   💼 경력 항목
   - 총 경력: [5] 년

   📝 코칭 분야 이력
   - 비즈니스코칭: [3년간 중소기업 대표 대상...]
   - 증빙: [파일 선택] (선택사항)

   ✍️ 자유 응답
   - "귀하의 코칭 철학은?"
   - [텍스트 입력...]

4️⃣ 임시저장 (선택)
   - 작성 중 "임시저장" 버튼
   - 나중에 이어서 작성 가능

5️⃣ 제출 전 확인
   - 필수 항목 모두 작성했는지 자동 확인
   - 증빙 파일 첨부 확인
   - "제출하기" 버튼 클릭

6️⃣ 제출 완료
   - 제출 완료 메시지 확인
   - 이메일 알림 수신 확인
```

**체크포인트**:
- [ ] 어떤 항목을 입력해야 하는지 명확한가?
- [ ] 복수 입력 (자격증 추가)이 쉬운가?
- [ ] 파일 업로드가 간편한가?
- [ ] 임시저장 기능이 유용한가?
- [ ] 필수 항목 검증이 제대로 작동하는가?

**예상 소요 시간**: 약 30분

```

│                                                      │
│ 자격증 2                                [❌ 삭제]    │
│ ┌──────────────────────────────────────────────┐   │
```

1. 첫 번째 자격증 정보 입력
2. "자격증 추가" 버튼 클릭
3. 두 번째 자격증 정보 입력
4. 최대 5개까지 추가 가능
5. "삭제" 버튼으로 제거 가능

### 4.3 파일 업로드

**지원 파일**:
- PDF, JPG, PNG, DOC, DOCX
- 최대 10MB

**업로드 방법**:
1. "파일 선택" 버튼 클릭
2. 파일 탐색기에서 파일 선택
3. 업로드 진행 상태 확인
4. 완료되면 파일명 표시

**주의사항**:
- ⚠️ 증빙이 "필수"인 경우 반드시 파일 첨부
- ⚠️ 파일 크기 초과 시 오류 메시지
- ⚠️ 지원하지 않는 파일 형식 주의

## 5. 피드백 제출

### 5.1 피드백 항목

다음 항목에 대해 자유롭게 의견을 주세요:

**1. 사용성**
- 어려운 부분이 있었나요?
- 설명이 부족한 부분이 있었나요?
- 더 편리했으면 하는 부분이 있나요?

**2. 성능**
- 느린 화면이 있었나요?
- 오래 걸리는 작업이 있었나요?

**3. 오류**
- 에러 메시지를 본 적이 있나요?
- 버튼이 작동하지 않은 적이 있나요?
- 데이터가 사라진 적이 있나요?

**4. 기능**
- 필요한데 없는 기능이 있나요?
- 불필요한 기능이 있나요?
- 개선이 필요한 기능이 있나요?

**5. 전반적 만족도**
- 1~10점 중 몇 점을 주시겠어요?
- 동료에게 추천하시겠어요?

### 5.2 피드백 제출 방법

**방법 1: 시스템 내 피드백 버튼**
- 우측 하단 "💬 피드백" 버튼 클릭
- 간단한 폼 작성 후 제출

**방법 2: 구글 설문지**
```
https://forms.gle/xxxxx (별도 제공)
```

**방법 3: 이메일**
```
feedback@yourdomain.com
```

**방법 4: 정기 미팅**
- 매주 금요일 14:00
- Zoom 링크: (별도 제공)

**좋은 피드백**:

## 6. FAQ

### Q1. 비밀번호를 잊어버렸어요
**A**: 로그인 화면에서 "비밀번호 찾기" 클릭 → 이메일로 재설정 링크 발송

### Q2. 파일을 업로드했는데 사라졌어요
**A**: 브라우저를 새로고침하면 임시저장된 내용이 사라질 수 있습니다. 작성 중에는 "임시저장" 버튼을 자주 눌러주세요.

### Q3. 제출 버튼이 비활성화되어 있어요
**A**: 필수 항목을 모두 작성했는지 확인해주세요. 빨간색 별표(*)가 있는 항목을 모두 입력해야 합니다.

### Q4. 모바일에서 파일을 업로드할 수 없어요
**A**: 일부 모바일 브라우저에서는 제한이 있을 수 있습니다. PC 사용을 권장합니다.

### Q5. 점수가 맞지 않아요
**A**: 자동 채점은 기준에 따라 부분 점수를 부여합니다. 최종 점수는 관리자가 수동으로 조정할 수 있습니다.

### Q6. 제출한 내용을 수정하고 싶어요
**A**: 제출 후에는 수정이 불가능합니다. 관리자에게 문의해주세요.

### Q7. 시스템이 느려요
**A**:

- 다른 브라우저를 사용해보세요
- 인터넷 연결을 확인해주세요
- 계속 문제가 있으면 피드백으로 제보해주세요
